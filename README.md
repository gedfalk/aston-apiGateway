## Aston - API Gateway

Часть микросервисного RESTful Spring Boot приложения.
Единая точка входа для микросервисов:
- user-service
- notification-service

Помимо API Gateway прописаны контроллеры для Circuit Breaker.

Запуск через [aston-serviceOrchestration](https://github.com/gedfalk/aston-serviceOrchestration)
